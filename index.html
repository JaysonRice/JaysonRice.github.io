Frame Advance - C#, .NET, SQL Server, React

  • Final capstone at NSS; A full stack application for people who play competitive games to review their videos easier
  • Built using React framework, an ASP.NET Web API and SQL Server

Frame Advance

Frame Advance was built with competitors in mind. I personally compete in Super Smash Bros Melee tournaments, and the best way to improve is generally to analyze your past tournament games. Users can create a post with a YouTube link and create timestamps of notable moments in the video to write notes about. This way, users will have a convenient database of their own notes with video reference available.

Users can also find reviews by other people if they don't want to review their own footage to learn.

Trivia Training - HTML, CSS, JS, React.js

  • Front-end React app that presents the user 10 multiple choice questions per a round for them to practice.
  • Questions are randomized and the user is presented with a new score for every round.

Trivia Training

Visit the Deployed App

This was a code challenge issued by Tandem to apply for one of their Apprenticeship Programs. Entrants had a week to submit their code using any language they prefer.

Application Requirements

  • A round of trivia has 10 Questions
  • All questions are multiple-choice questions
  • Questions should not repeat in a round.
  • Questions with their multiple choice options must be displayed one at a time.
  • A user can select only 1 answer out of the 4 possible answers.
  • The correct answer must be revealed after a user has submitted their answer.
  • A user can see the score they received at the end of the round
